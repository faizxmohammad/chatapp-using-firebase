# chatapp-using-firebase


To use the app :
1.Fork this repository .</br></br>
2.Download this repository ,extract , open it with android studio (or) simply open your android studio select get from VCS and clone it by pasting this forked repository link.</br></br>
3.Delete the already present google-services.json file present in ChatsApp\app folder and add your google-serives.json file in it.</br></br>
4.Activate your Realtime databases, storage , phone authentiaction through the firebase console. </br></br>
5.Volia! your Chatting application is ready to use :)
